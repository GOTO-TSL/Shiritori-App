# ShiritoriApp
英単語しりとりゲームアプリです</br>
敵が繰り出してきた英単語にしりとりで返すと敵にダメージを与えることができます．</br>
敵を倒すとゲームクリアです．</br>
ゲームで使用した単語を単語帳に保存して確認することができます．

しりとりのルール
* 敵の繰り出す英単語の最後の文字から始まる英単語を入力すること
* 2文字以上の単語であること
* 存在する単語であること</br>

 
## Requirement
 
* GRDB 5.8.0
* Hero 1.6.1
